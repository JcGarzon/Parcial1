Cómo usar .gitignore 🤖

1. Crear el archivo: En la carpeta principal de tu proyecto, crea un archivo llamado .gitignore.

2. Añadir archivos a ignorar: Escribe dentro del archivo los nombres de los archivos o carpetas que no quieres que Git rastree.

3. Confirmar que Git los ignora: Usa git status para verificar que los archivos listados no aparecen como cambios pendientes.

4. Actualizar el archivo: Si necesitas ignorar más archivos, solo edita .gitignore y agrégalos.
